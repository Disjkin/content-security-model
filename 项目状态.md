## 技术决策记录

| 日期 | 决策 | 理由 |
|------|------|------|
| 2026/06/10 | 使用 Trie 树作为敏感词匹配算法 | O(m) 查找复杂度，适合大规模词库 |
| 2026/06/10 | FastAPI 作为 Web 框架 | 异步高性能，自带 OpenAPI 文档 |
| 2026/06/10 | TF-IDF + 分类模型作为推理引擎 | 轻量级、可训练、无需下载大模型 |
| 2026/06/10 | 敏感词 + 模型双层检测策略 | 兼顾速度与语义理解能力 |
| 2026/06/10 | scikit-learn 作为模型训练框架 | 成熟稳定，模型文件小，部署方便 |
| 2026/06/10 | SQLite 作为日志存储 | 零依赖、单文件、适合中小规模 |
| 2026/06/10 | 异步队列刷写日志 | 避免 I/O 阻塞检测主路径 |
| 2026/06/10 | ThreadPoolExecutor 批量并行 | 混合 I/O+CPU 负载下有效提升吞吐 |
| 2026/06/10 | 原始文本仅存 hash | 隐私保护，不持久化用户文本 |
| 2026/06/10 | 纯前端 SPA 管理界面 | 零构建工具，部署简单 |
